Okular-19.12.2

Introduction to Okular

Okular is a document viewer for KDE. It can view documents of many types including PDF, PostScript, TIFF, Microsoft CHM, DjVu, DVI, XPS and ePub.

This package is known to build and work properly using an LFS-9.1 platform.

Package Information

  • Download (HTTP): http://download.kde.org/stable/release-service/19.12.2/src/okular-19.12.2.tar.xz

  • Download MD5 sum: fbabf5da9faa4ab507cf37ae426f23d3

  • Download size: 6.9 MB

  • Estimated disk space required: 63 MB

  • Estimated build time: 1.2 SBU (using parallelism=4)

Okular Dependencies

Required

KDE Frameworks-5.67.0,

Recommended

libkexiv2-19.12.2, LibTIFF-4.1.0, and Poppler-0.87.0 (built with Qt5, required for PDF support)

Optional

qca-2.3.0, DjVuLibre, kpimtextedit, libspectre (for PostScript support), libchm, libepub, and Mobipocket

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/okular5

Installation of Okular

Install Okular by running the following commands: 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

This package does not come with a test suite.

Now, as the root user: 

make install
