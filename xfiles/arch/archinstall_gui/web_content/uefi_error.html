ArchLinuxinstaller {version}

Archinstall only supports UEFI

Please reboot with UEFI enabled.

VirtualBox

For VirtualBox this is done by:
  • Machine settings
  • System
  • Tick/check "Enable EFI (special OSes only)"
Important note: ArchLinux (and others) boot extremely slow (2min boot time) when turning EFI on in VirtualBox.

VMWare

  • Virtual Machine settings
  • Advanced
  • Tick/check "Boot with EFI instead of BIOS"
Under MAC, this is under "VM Options" and them "Boot Options" and "Firmware".

KVM

For KVM, this is a bit more tricky.
Most likely you'll use a GUI for this, and if you're using libvirt then virt-install has an option for this.

If you're using quemu-system-x86_64 as a standalone, Here's a script that enables UEFI.
